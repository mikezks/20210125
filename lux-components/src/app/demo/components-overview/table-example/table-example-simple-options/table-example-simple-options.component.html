luxMultiSelect Ermöglicht es, diese Tabelle mit Multiselection-Funktionalitäten zu versehen. luxNoDataText Diese Property enthält den Text der angezeigt wird wenn keine Datensätze gefunden werden konnten. luxShowFilter Mit diesem Flag lässt sich das Filter-Inputfeld aus-/einblenden. luxFilterText Diese Property definiert das Label des Filter-Inputfelds.
luxShowPagination Diese Property aktiviert/deaktiviert die Pagination für die Tabelle. luxPageSize Hiermit kann die aktuelle Anzahl an Einträgen für die Pagination eingestellt werden. luxPageSizeOptions Diese Property enthält ein Array mit den möglichen Optionen für die Anzahl an Einträgen.
